Mission

The web is full of simple tasks that should be simple to do. Convert a file. Preview a document. Generate a QR code. Instead, we're asked to create accounts, accept tracking, watch ads, or pay subscriptions for things that should be free.

Tool Commons is a directory of tools built differently. Each tool does one thing well. Each is open source. Each runs in your browser without sending your data anywhere. Each is built by an individual developer who wanted the tool to exist.

To be listed here, a tool must meet these criteria:

  • ✓ Open source — Code is public under an OSI-approved license
  • ✓ No login required — Core functionality works without creating an account
  • ✓ Local-first — Your data stays in your browser unless you explicitly export it
  • ✓ No ads, no tracking — Privacy-respecting analytics only (aggregate, anonymous)
  • ✓ Static hostable — Can run from GitHub Pages or similar free hosting
  • ✓ Single purpose — Does one thing well rather than everything poorly
